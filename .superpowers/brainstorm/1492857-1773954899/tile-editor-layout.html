Tile Editor Layout

How should the hex grid sit within the campaign admin page?

A

Grid + Table side by side

The hex grid sits above or beside the tile list. Click an empty hex to open a "create tile" form. Click an existing hex to select it in the table below for editing.

B

Grid is the primary interface

The hex grid replaces the tile table entirely. Click any hex to open an inline property panel. Empty hexes show a "+" to create. No separate list view.

C

Grid for creation only, table for editing

An "Add Tile" section shows the grid (only empty slots visible). Editing existing tile properties stays in the existing table rows (inline edit on click, as the team column works now).
