Admin UI Structure

Pages and navigation within the /admin section of the Vite app

Pages by role

Any logged-in user
  • /admin — dashboard (role-aware)
  • /admin/login — OAuth login buttons
  • /admin/campaigns/new — create campaign
GM (per campaign)
  • /admin/campaigns/{id} — campaign overview
  • /admin/campaigns/{id}/map — tile ownership editor
  • /admin/campaigns/{id}/attacks — manage attacks
  • /admin/campaigns/{id}/teams — teams + assets
  • /admin/campaigns/{id}/members — manage players
Superuser
  • /admin/users — all users + role management
  • Plus all GM pages across all campaigns

Player pages

Player (scoped to their team)
  • /admin/campaigns/{id}/teams/{tid} — edit own team profile
  • /admin/campaigns/{id}/attacks — submit / cancel own attacks

Map editor approach

A

Simple 2D grid form

A table/grid showing all tiles. Click a tile row to change its owner via a dropdown. Fast to build, no 3D dependency.

B

Reuse the 3D Babylon.js map

Embed the existing map viewer with an "edit mode" overlay. Click a tile on the 3D map to open an edit panel. More immersive but more complex.
